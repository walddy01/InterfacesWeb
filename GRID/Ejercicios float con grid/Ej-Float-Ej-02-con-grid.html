Otra forma de hacerlo, más fea pero válida, sería con un contenedor interior:

HTML: 
<div id="container">

  <div id="container-interior">
    <div class="div2">Div2</div>
    <div class="div3">Div3</div>
  </div>

CSS: 
#container {
  display: flex;
  flex-direction: row-reverse;
  justify-content: space-between;
}

#container-interior {
  display: flex;
}
